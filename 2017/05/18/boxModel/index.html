css中 Box model 分类

css中Box 分为两种，一种是W3C文章 和 IE 标准盒子模型

怪异模式是“部分浏览器在支持w3c标准的同时还保留了原来的解析模式，怪异模式主要表现在IE内核的浏览器

在标准模式下，一个块的总宽度 = magin + padding + border

在怪异模式下，一个快的总宽度 = width + margin(也就是说width已经包含了padding和border值)

css中的box-sizing属性

box-sizing的语法：

当设置为box-sizing:content-box时，将采用标准模式解析计算，也是默认模式；

当设置为box-sizing:border-box时，将采用怪异模式解析计算；

说的简单点的话就是，如果想让设置padding或者border但是不影响盒子本身的宽度的话，那就设置box-sizing:border-box ,这样就是不会让padding或者 border的值影响盒子大小了
