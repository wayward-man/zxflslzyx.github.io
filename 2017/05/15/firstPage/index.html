markdown 基本操作

关于段落换行

单个回车
视为空格。
连续回车

才能分段。

行尾加两个空格，这里->  
即可段内换行。

*这些文字显示为斜体*

**这些文字显示为粗体

关于标题

标题的话用#就行了，1-6个#分别代表1-6级标题，其中一个#代表最大字体

示例： 

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

###输出结果为

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

ps：一定要在#后面加一个空格，否则可能不行，自己的亲测，别人的编辑器就不清楚了

显示代码

直接在前面输入四个空格即可
比如这样

代码：

[四个空格]function lala(){
var a =”just test”;
}

效果就是这样的

    function lala(){
     var a ="just test";
}

代码：
