boxModel

css中 Box model 分类

css中Box 分为两种，一种是W3C文章 和 IE 标准盒子模型

怪异模式是“部分浏览器在支持w3c标准的同时还保留了原来的解析模式，怪异模式主要表现在IE内核的浏览器

在标准模式下，一个块的总宽度 = magin + padding + border

在怪异模式下，一个快的总宽度 = width + margin(也就是说width已经包含了padding和border值)
